Playful Interacion Topic

Het vierde en laatste onderwerp wat wij hebben behandeld bij HCI is playful interaction. Playful interaction is een techniek waar gebruikers op een speelse manier een actie kunnen voltooien. Playful interaction kan op heel veel plekken worden toegepast om van een saai proces een heel leuk proces te maken. Bijvoorbeeld in een museum is het voor kinderen veel leuker om via een digitaal (interactief) spel kennis te vergaren, dan te kijken naar een muur tekst en een plaatje.

2f

Voor opdracht 2f heb ik verschillende vormen die verschillende groottes en kleuren hebben neerzetten in mijn programma. Dit is de eerste opdracht van Computer Vision dus ik zal alle code uitleggen. size(200,200) zorgt ervoor dat er een canvas komt die 200 bij 200 pixels is. Hier staat er ellipseMode, deze functie bepaald waar de cirkel terecht komt. Hierna staat er fill(), dit zorgt ervoor dat de cirkel een kleur krijgt. noStroke() zorgt ervoor dat de cirkels een uitlijning hebben. ellipse() is de cirkel zeld, waar de waardes te maken hebben met de plaatsing en de hoogte en breedte.

2i

Voor opdracht 2i heb ik nog meer verschillende vormen moeten maken. De code lijkt heel erg op wat er bij opdracht 2f gebeurt, alleen in plaats van ellipse zijn er ook triangle, rect en quad. Ook staan er meer waardes bij om te bepalen waar de vorm staat en hoe groot en breed deze is.

2o

Voor opdacht 2o moest ik een cirkel maken die de muis volgde, en een lijntje die aan deze cirkel vast zat. De nieuwe code die hier opvalt zijn de functies mouseX en mouseY. Deze zorgen ervoor dat de code in de gaten houdt waar mijn muis is, en deze waardes gebruikt het om de cirkel ergens op het canvas te plaatsen.

3b

Voor opdracht 3b moest ik Processing koppelen aan Arduino. Hier moest ik een script maken dat wanneer mijn muis over de vorm ging, er een LED aangaat op mijn Arduino. Het Arduino script ziet er heel bekend uit van de Arduino workshop, dus voor uitleg van deze code kan je het beste naar Wearables of Artificial Creatures. De processing code is ook vrij simpel. In de void setup() staat code om te verbinden met de Arduino. Hierna in void draw() staat een if statement om te controlleren of mijn muis in de cirkel zit, en als deze in de cirkel zit stuurt de code de letter H naar de Arduino. Daardoor weet de Arduino dat het de LED uit moet zetten door een if-statement.

Topic assignment

Na de computer vision opdrachten zijn er drie assignment. De eerste assignment is een opdracht om een playful interaction concept te bedenken voor de Haagse Hogeschool. De tweede assignment is om research te doen naar al bestaande oplossingen van playful interaction. De derde assignment is om te reflecteren op dit onderwerp.

Assignment A: A new playful interaction concept

Voor assignment A ga ik een concept bedenken voor een sociale playful interface in de Haagse Hogeschool om positief gedrag aan te sporen.

In de Haagse Hogeschool komen er boven de prullenbakken interface te hangen. Op deze interfaces staat een score of afval goed is afgescheiden en dus in de correcte prullenbak is gedaan. Via sensoren in de prullenbak weet de prullenbak of het goed is weggegooit of niet. Wanneer iemand iets goed weggooit krijgt deze positieve feedback op het interface te zien, en komt er een punt bij. Wanneer er boven de 80% goed wordt gescheiden dan is de interface groen en staan er positieve berichten. Wanneer het onder de 80% is is de interface rood en staan er tips hoe je correct moet scheiden.

Assignment B: Research

Voor deze opdracht doe ik onderzoek naar twee projecten die playful interaction inzetten. Het eerste project wat ik heb gekozen is “Crosswalk That Lets You Play Pong With A Stranger On The Other Side”. Dit is een installatie gebouwd in Duitsland door team van creatieve design studenten. De installatie is zoals de titel zegt een Zebrapad waarmee je pong kunt spelen met een vreemdeling aan de andere kant. Wanneer iemand staat te wachten op wanneer het licht op groen springt kunnen ze met een vreemdeling aan de overkant een potje pong spelen totdat het licht op groen springt. Dit hebben ze gemaakt zodat het wachten op een groen stoplicht niet meer zo saai aanvoelt. De installatie werkt doordat er twee spelers gaan spelen. Door de input van beide kan het spel gespeeld worden. De installatie is structured, multiplayer, competitive en heeft interdependent acties.

Het tweede project waar ik onderzoek naar heb gedaan is de tentoonstelling van het Museon over One Planet. Als uitgangspunt voor de tentoonstelling heeft het Museon de Duurzame Ontwikkelingsdoelen genomen. Dit zijn zeventien doelen die de Verenigde Naties voor 2030 willen bereiken. In de tentoonstelling heeft ieder ontwikkelingsdoel een eigen eiland. Op dit eiland is er een uniek spel die te maken heeft met het ontwikkelingsdoel. Ook is er een strippenkaart met allemaal vraagstukken erop. Deze vragen kun je beantwoorden bij ieder eiland en zo kan je je hele strippenkaart vol maken.

Deze tentoonstelling is een gigantische playful interaction installatie. Ieder eiland heeft een ander spel waar je spelenderwijs leert over de ontwikkelingsdoelen die de VN heeft opgesteld. Op deze manier zorgt het Museon ervoor dat de nieuwe generaties op een leuke manier over deze doelen leren.

Assignment C: Reflection

Wat vond je van het topic? Playful interaction is een heel interessant onderwerp. Er zijn heel veel toepassingen die super tof zijn en die ik stiekem zelf wel wil uitproberen. Ook de workshops waren hartstikke leuk en ik heb toffe opdrachten gemaakt.

Hoe zie je applicaties en de toekomst van dit? Ik zie hier wel een toekomst in. Veel mensen vinden dit super interessant en zouden hier ook meer tijd in willen steken. Het maakt het leven een stuk leuker om playful interaction toe te passen. Ook kunnen er praktische oplossingen komen met playful interaction die de wereld een stuk beter kunnen maken (bijvoorbeeld mijn concept met afvalscheiding).

Heeft het topic je verwachtingen tegemoet gekomen? Zeker wel. Ik wist in eerste instantie niet helemaal wat ik moest verwachten, maar ik vond het echt een leuk onderwerp en zou hier later ook echt wel wat mee willen doen.
